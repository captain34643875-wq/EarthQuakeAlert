# 지진 속보 Chrome 확장 프로그램

전세계 최신 지진 정보를 실시간으로 수집하여 알려주는 Chrome 확장 프로그램입니다.

## 주요 기능

### 🌍 다국적 지진 데이터 수집
- **USGS** (미국 지질조사소) - 전 세계 지진
- **KMA** (한국 기상청) - 한국 지진
- **JMA** (일본 기상청) - 일본 지진  
- **EMSC** (유럽-지중해 지진 센터) - 유럽 및 지중해 지진

### 🔄 실시간 업데이트
- 20초마다 자동으로 최신 지진 정보 확인
- 새 지진 발생 시 즉시 알림 표시
- 최근 24시간 내 지진 정보만 표시

### 🎨 규모별 색상 구분
- **M6 이상**: 🔴 빨간색 (강한 지진)
- **M4~6**: 🟠 주황색 (중간 지진)
- **M4 미만**: 🟡 노란색 (약한 지진)

### 📋 복사 기능
- "전체 복사하기" 버튼으로 지진 정보를 클립보드에 복사
- 형식: `[지진속보] 진원지: 위치 / 추정규모: M5.2 / 예상최대진도: 진도 5-6`

### 📊 상세 정보 표시
- 위치, 규모, 깊이, 발생 시간
- 데이터 출처 (어느 기관에서 가져왔는지)
- 좌표 정보 (위도, 경도)

## 설치 방법

### 1. 소스코드 다운로드
```bash
git clone [저장소 주소]
cd EarthQuakeAlert/extension
```

### 2. Chrome 확장 프로그램 로드
1. Chrome 브라우저에서 `chrome://extensions/` 주소로 이동
2. 우측 상단의 "개발자 모드" 활성화
3. "압축해제된 확장 프로그램을 로드합니다" 클릭
4. `extension` 폴더 선택

### 3. 권한 확인
확장 프로그램이 다음 권한을 요청합니다:
- `alarms`: 주기적 데이터 확인
- `notifications`: 새 지진 알림
- `storage`: 지진 데이터 저장
- 외부 API 접근 (지진 데이터 수집)

## 사용 방법

### 기본 사용
1. 확장 프로그램 아이콘 클릭하여 팝업 열기
2. 최근 지진 목록 확인
3. 새 지진은 빨간색으로 강조 표시

### 새로고침
- "새로 고침" 버튼으로 즉시 데이터 업데이트

### 복사하기
1. "전체 복사하기" 버튼 클릭
2. 지진 정보가 클립보드에 복사됨
3. 원하는 곳에 붙여넣기 사용

## 기술 사양

- **Manifest Version**: V3 (최신 Chrome 확장 표준)
- **데이터 수집 주기**: 20초
- **저장 데이터**: 최근 20개 지진 정보
- **중복 제거**: 시간(5분), 위치(1도), 규모(0.5) 기준
- **알림 우선순위**: 높음 (priority: 2)

## API 정보

### USGS (미국 지질조사소)
- 엔드포인트: `https://earthquake.usgs.gov/earthquakes/feed/v1.0/summary/all_hour.geojson`
- 데이터 형식: GeoJSON
- 제공 범위: 전 세계

### KMA (한국 기상청)
- 엔드포인트: `https://api.weather.go.kr/weather/earthquake/list`
- 데이터 형식: XML/JSON
- 제공 범위: 한국 및 주변

### JMA (일본 기상청)
- 엔드포인트: `https://www.jma.go.jp/bosai/quake/data/list.json`
- 데이터 형식: JSON
- 제공 범위: 일본 및 주변

### EMSC (유럽-지중해 지진 센터)
- 엔드포인트: `https://www.emsc-csem.org/service/api/query`
- 데이터 형식: JSON
- 제공 범위: 유럽 및 지중해

## 주의사항

1. **API 제한**: 일부 기관의 API는 사용량 제한이 있을 수 있습니다
2. **데이터 정확성**: 각 기관의 데이터 수집 시점이 달라 완전히 동기화되지 않을 수 있습니다
3. **예상 진도**: 규모를 기반으로 한 단순 추정치이며, 실제 진도와 다를 수 있습니다
4. **네트워크**: 인터넷 연결이 필요하며, 네트워크 상태에 따라 데이터 수집이 지연될 수 있습니다
