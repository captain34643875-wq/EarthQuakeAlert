# 지진 속보 Chrome 확장 프로그램

전세계 최신 지진 정보를 실시간으로 수집하여 알려주는 Chrome 확장 프로그램입니다.

## 주요 기능

### 다국적 지진 데이터 수집
- **USGS** (미국 지질조사소) - 전 세계 지진 ✅ 현재 지원
- **EMSC** (유럽-지중해 지진 센터) - 유럽 및 지중해 지진 🔄 안정화 후 재활성화 예정

### 실시간 알림 및 UI
- **20초마다 자동 업데이트**
- **새 지진 발생 시 즉시 알림**
- **규모별 색상 구분**: M6 이상, M4~6, M4 미만
- **복사 기능**: `[지진속보] 진원지: 위치 / 추정규모: M5.2 / 예상최대진도: 진도 5-6`

### 상세 정보 표시
- 위치, 규모, 깊이, 발생 시간
- 데이터 출처 표시 (어느 기관에서 가져왔는지)
- 좌표 정보 (위도, 경도)

## 🚀 설치 방법

### 1. Chrome 확장 프로그램 로드
1. Chrome 브라우저에서 `chrome://extensions/` 주소로 이동
2. 우측 상단의 **"개발자 모드"** 활성화
3. **"압축해제된 확장 프로그램을 로드합니다"** 클릭
4. `extension` 폴더 선택

### 2. 권한 확인
확장 프로그램이 다음 권한을 요청합니다:
- `alarms`: 주기적 데이터 확인
- `notifications`: 새 지진 알림
- `storage`: 지진 데이터 저장
- 외부 API 접근 (지진 데이터 수집)

## 🎯 사용 방법

### 기본 사용
1. 확장 프로그램 아이콘 클릭하여 팝업 열기
2. 최근 지진 목록 확인
3. 새 지진은 빨간색으로 강조 표시

### 새로고침
- **"새로 고침"** 버튼으로 즉시 데이터 업데이트

### 복사하기
1. **"전체 복사하기"** 버튼 클릭
2. 지진 정보가 클립보드에 복사됨
3. 원하는 곳에 붙여넣기 사용

## 🔧 기술 사양

- **Manifest Version**: V3 (최신 Chrome 확장 표준)
- **데이터 수집 주기**: 20초
- **저장 데이터**: 최근 20개 지진 정보
- **중복 제거**: 시간(5분), 위치(1도), 규모(0.5) 기준
- **알림 우선순위**: 높음 (priority: 2)

## 🌐 API 정보

### USGS (미국 지질조사소)
- 엔드포인트: `https://earthquake.usgs.gov/earthquakes/feed/v1.0/summary/all_hour.geojson`
- 데이터 형식: GeoJSON
- 제공 범위: 전 세계
- 상태: ✅ 현재 지원 중

### EMSC (유럽-지중해 지진 센터)
- 엔드포인트: `https://www.seismicportal.eu/fdsnws/event/1/query`
- 데이터 형식: GeoJSON
- 제공 범위: 유럽 및 지중해
- 상태: 🔄 안정화 후 재활성화 예정

### KMA (한국 기상청)
- 엔드포인트: `https://api.weather.go.kr/weather/earthquake/list`
- 데이터 형식: XML/JSON
- 제공 범위: 한국 및 주변
- 상태: 🔄 추후 추가 예정 (API 키 필요)

### JMA (일본 기상청)
- 엔드포인트: `https://www.jma.go.jp/bosai/quake/data/list.json`
- 데이터 형식: JSON
- 제공 범위: 일본 및 주변
- 상태: 🔄 추후 추가 예정 (데이터 파싱 필요)

## ⚠️ 주의사항

1. **API 제한**: 일부 기관의 API는 사용량 제한이 있을 수 있습니다
2. **데이터 정확성**: 각 기관의 데이터 수집 시점이 달라 완전히 동기화되지 않을 수 있습니다
3. **예상 진도**: 규모를 기반으로 한 단순 추정치이며, 실제 진도와 다를 수 있습니다
4. **네트워크**: 인터넷 연결이 필요하며, 네트워크 상태에 따라 데이터 수집이 지연될 수 있습니다
